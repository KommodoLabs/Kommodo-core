# Kommodo-core
Kommodo core smart contracts.

## Install Dependencies
```
npm i
forge install foundry-rs/forge-std
uv tool install --python 3.12 halmos
```

## Compile
```
npm run compile
```

## Security

### Tests
```
npm run test
```

### Fuzz test
```
npm run fuzz
```

### Formal test
```
npm run formal
```
